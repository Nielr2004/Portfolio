## ğŸš€ Getting Started

Follow these steps to get a local copy up and running.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1.  Clone the repository:
    ```bash
    git clone [Your Repository URL]
    ```
2.  Navigate into the project directory:
    ```bash
    cd [your-portfolio-repo-name]
    ```
3.  Install dependencies:
    ```bash
    npm install
    # or yarn install
    # or pnpm install
    # or bun install
    ```

### Running Locally

To start the development server:

```bash
npm run dev
# or yarn dev
# or pnpm dev
# or bun dev
````

This will typically open the application in your browser at `http://localhost:5173` (or another port if 5173 is in use).

### Building for Production

To create a production-ready build of your application:

```bash
npm run build
# or yarn build
# or pnpm build
# or bun build
```

The optimized build files will be located in the `dist` folder.
